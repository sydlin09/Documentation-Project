The Basic Stitches

Learn which stitch to use for different repairs.

Ladder Stitch:

  • Moves in a simple up-and-down pattern.

  • It is useful for attaching patches and making temporary repairs.

Back Stitch:

  • Overlaps previous stitches, creating a stronger seam.

  • It is recommended for jeans, heavy fabrics, and long-lasting repairs.

  • A stronger stitch where you go "one step back" before moving forward; best for heavy denim.
